Política de Privacidade

1. Informações Coletadas

São coletadas apenas as informações necessárias para vincular TikTok ao Discord:

  • Nome de usuário do TikTok
  • ID do Discord

2. Uso das Informações

Usadas exclusivamente para verificar se o usuário é membro e dar/remover cargos no Discord.

3. Compartilhamento

Nenhum dado é compartilhado com terceiros.

4. Exclusão de Dados

O usuário pode solicitar exclusão a qualquer momento.

5. Aceitação

Ao usar o app, você concorda com esta política.
